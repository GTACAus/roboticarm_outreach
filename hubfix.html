GTAC – SPIKE to Robot Inventor Firmware Recovery Tool

⚠️ This tool must be run as Administrator on a Windows computer. Right-click the downloaded file and select Run as Administrator.

If a LEGO Hub has been accidentally restored to SPIKE firmware, this tool will return it to LEGO Mindstorms Robot Inventor firmware so it can be used again in the Mindstorms Inventor App.


How to Use

  1. Download the tool using the button above.
  2. Right-click fix.bat and choose Run as Administrator.
  3. Follow the on-screen instructions carefully.
  4. When prompted, put the hub into bootloader (colour cycling) mode:
    • Turn hub off completely.
    • Hold the Bluetooth button.
    • Plug in USB while still holding the button.
    • Continue holding until colours cycle RED → GREEN → BLUE → PURPLE.
  5. Follow the Zadig driver installation steps when they appear.
  6. Wait for flashing to complete.

Important Notes

  • If the hub does not appear in Zadig after clicking "List All Devices", your computer likely already has the correct WinUSB driver installed. Close Zadig and continue.
  • If LEGO Mindstorms Inventor later displays a motor update required message, ignore this and continue. It does not affect hub firmware restoration.

Troubleshooting

Hub not appearing in Zadig Ensure the hub is in colour-cycling bootloader mode. If it still does not appear, the driver may already be installed — close Zadig and continue.
Flash failed message Confirm:
  • Hub is in colour cycling mode
  • USB cable supports data (not charge-only)
  • Driver was installed correctly
Restart the process if needed.
Windows SmartScreen warning Click “More info” → “Run anyway”. This appears because the batch file is not digitally signed.
Python installation seems stuck Close the window and run the tool again. The script automatically installs Python if required.
Hub still not recognised in Inventor app Unplug the hub, restart the computer, and reconnect the hub before reopening the Inventor app.
